Bienvenue

Si vous êtes fan du web comme Tim Berners-Lee, ce site vous permettra de vous quelque mémo du web. Notamment certaines balises et pour aller loin, on vous propose aussi de vous remémorier les dates clés du WEB.

C'est quoi le WEB ?

Le Web eest tout sinplement un système d'informations, dynamique en plus multi-plateformes qui tourne sur internet, elle a été proposé en 1989 par Tim Berners-Lee. Le Web nous aide à gérer de l'information a les liers et à les partager

HTML

Html est un langage de crétation de site Web. Sous le nom de Hypertext markup Language, il est utilisé pour representer le contenu d'une page WEB en utilisant des balises. Sa premiere version est sorti en 1989 à la création du Web.
Ensuite, en 1994 HTML2 fait son appartion avec la possibilité d'insérer des images et des formulaires dans nos pages HTML. Puis en 1996 la version 3 sort avec la possibilité d'insere des son puis des video en 1998.
Aujourd'hui toute les plateformes doiventetre sur la version HTML 5.1.

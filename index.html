Bienvenue

Si vous êtes fan du web comme Tim Berners-Lee, ce site vous permettra de vous donner mémo du web. Notamment certaines balises et pour aller loin, on vous propose aussi de vous remémorier les dates clés du WEB.

C'est quoi le WEB ?

Le Web est tout sinplement un système d'informations, dynamique et multi-plateformes qui tourne sur internet, il a été conçu en 1989 par Tim Berners-Lee. Le Web nous aide à gérer de l'information, à les lier, et à les partager.

HTML

Html est un langage de création de sites Web. Sous le nom de Hypertext markup Language, il est utilisé pour representer le contenu d'une page WEB en utilisant des balises. Sa première version est sortie en 1989 à la création du Web.
Ensuite, en 1994 HTML2 fait son apparition avec la possibilité d'insérer des images et des formulaires. Puis en 1996, la version 3 sort avec la possibilité d'inserer des sons, puis des vidéos en 1998.
Aujourd'hui toutes les plateformes doivent être sur la version HTML 5.1.

CSS

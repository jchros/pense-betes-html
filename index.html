Bienvenue

Si vous êtes fan du web comme Tim Berners-Lee, ce site vous permettra de vous donner des mémos du web. Notamment certaines balises et pour aller plus loin, on vous propose aussi de vous remémorer les dates clés du Web.

C'est quoi le Web ?

Le World Wide Web est tout simplement un système d'information, dynamique et multi-plateforme qui tourne sur Internet, il a été conçu en 1989 par Tim Berners-Lee. Le Web nous aide à gérer de l'information, à les lier, et à les partager.

HTML

L'HTML est un langage de création de sites Web. Il est utilisé pour représenter le contenu d'une page Web en utilisant des balises. Sa première version est sortie en 1989 à la création du Web.
Ensuite, en 1994 HTML2 fait son apparition avec la possibilité d'insérer des images et des formulaires. Puis en 1996, la version 3 sort avec la possibilité d'insérer des sons, puis des vidéos en 1998.
La version la plus récente est l'HTML 5.

CSS

Le CSS sert pour la mise en forme de notre page HTML. Sa version la plus récente est le CSS3.

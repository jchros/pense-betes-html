Les balises de présentation

b

Cette balise nous permettra de mettre en gras un mot clés ou un fragement de texte

Exemple : Voici du texte en <b>gras</b> soit très gras

code

La balise <code> nous permettra d'utiliser des fragements de code d'un langage, Elle lui permettra de lui donner une police spécifique par rapport au autre.

Pour noter du code utilisez la balise <code> soit <p>Hello<p>

em

Cette balise permet de mettre un élément d'un texte en évidence.

Exemple : Ce texte est <em>en évidence</em> comme ce texte en évidence

hr

Cette balise permet de representer une nouvelle idée d'un bloc de texte

On va voir comment on peut faire voler des avions ...

Maintenant comment on va faire voler la voiture

i

Cette balise perme de mettre en italique un fragement de texte

Exemple : Ce texte est en <i>italique</i> comme ce texte

p

Cette balise permet de définir et d'afficher un paragraphe.

Exemple : Ce paragraphe est en <p> Hello, j'apprend à coder </i> ici

Hello, j'apprend à coder

main

pre

La balise <pre>permet d'écrire du code plus long sans balises br:

          #include &lt;iostream&rt;

            int main () {
                std::cout << "Hello World!\n";
                return 0;
            }

La balise pre permet d'écrire du code plus long sans balises br:

          #include &lt;iostream&gt;

            int main() {
                std::cout << "Hello World!\n";
                return 0;
            }

strong

Elle permet d'indiquer qu'un élément d'un texte est très important

Exemple : Ce texte est <strong> très important </strong> comme cette exemple très important

sub

Cette balise permet de mettre en indice du texte(en bas et plus petit)

Exemple : Calculons x<sub>0</sub> = 1 soit x0 = 1

sup

Cette balise pemet de mettre du texte en exposant(en haut et plus petit)

Exemple : Calculons : 2<sup>3</sup> = 8 soit 23 = 8

u

Cette balise permet de souligné

Exemple : Ce texte est <u>souligné</u>

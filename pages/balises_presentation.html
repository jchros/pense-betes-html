Les balises de présentation

b
Cette balise nous permettra de mettre en gras un mot clé ou un fragment de texte.

Exemple : Voici du texte en <b>gras</b> soit gras

br
Cette balise insert un saut à la ligne.

Exemple : <p>Après cette virgule, <br/> il y aura un saut à la ligne</p>

Après cette virgule,
il y aura un saut à la ligne

canvas
Cette balise permet de contenir du contenu graphique créé avec JavaScript.
code
Cette balise nous permettra d'utiliser des fragments de code d'un langage. Elle nous permettra de lui donner une police spécifique par rapport aux autres.

Pour noter du code utilisez la balise <code> soit code

del
Cette balise permet de remplacer du texte par un autre sans pour autant supprimer l'ancien texte. Par défaut, cela est représenté en le barrant.

Exemple : HTML est <del>nul</del> <ins>génial</ins> devient HTML est nul génial

details
Cette balise permet de mettre du texte ne montrant qu'une partie de son contenu par défaut, et permet à l'utilisateur d'afficher le reste de son contenu.
em
Cette balise permet de mettre un élément d'un texte en évidence.

Exemple : Ce texte est <em>en évidence</em> comme ce texte en évidence

hr
Cette balise permet de représenter un changement de thème dans un bloc de texte.

Exemple :

On va voir comment on peut faire voler des avions ...

Maintenant comment on va faire voler la voiture

i
Cette balise permet de mettre en italique un fragment de texte.

Exemple : Ce texte est en <i>italique</i> comme italique

ins
Cette balise permet de représenter du texte qui remplace un autre (représenté par la balise <del>). Par défaut, cela est représenté en le soulignant.

Exemple : HTML est <del>nul</del> <ins>génial</ins> devient HTML est nul génial

mark
Cette balise permet de surligner du texte.

Exemple : <mark>surligné</mark> devient surligné.

p
Cette balise permet de définir et d'afficher un paragraphe.

Exemple : Ce paragraphe est en <p> Hello, j'apprend à coder </p> ici

Hello, j'apprend à coder

pre
La balise <pre> permet d'écrire du code plus long sans balises <br>

Exemple : <pre> #include &lt;iostream&rt; int main () { std::cout &lt;&lt; "Hello World!\n"; return 0; } </pre> 

            #include <iostream>

            int main() {
                std::cout << "Hello World!\n";
                return 0;
            }
small
Cette balise permet de mettre du texte en petite taille.

Exemple : <small>petit</small> devient petit

strong
Elle permet d'indiquer qu'un élément d'un texte est très important.

Exemple : Ce texte est <strong> très important </strong> comme cette exemple très important

sub
Cette balise permet de mettre en indice du texte(en bas et plus petit).

Exemple : Calculons x<sub>0</sub> = 1 soit x0 = 1

summary
Cette balise est utilisée en conjonction avec <details> en tant que contenu présenté avant que le reste du contenu soit dévoilé par l'utilisateur.
sup
Cette balise pemet de mettre du texte en exposant(en haut et plus petit).

Exemple : Calculons : 2<sup>3</sup> = 8 soit 23 = 8

u
Cette balise permet de souligner.

Exemple : Ce texte est <u>souligné</u> comme souligné

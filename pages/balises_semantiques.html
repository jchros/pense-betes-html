Les balises sémantiques

abbr
La balise <abbr> nous permet de définir une abréviation ou un acronyme elle n'est pas afficher dans la page html mais elle indique au navigateur la définition de cette abréviation

Exemple : <abbr title="Union nationale des entraîneurs et cadres techniques du football français">UNECATEF</abbr> soit UNECATEF

address
Cette balise permet de définir une adresse.
blockquote
Cette balise définit une citation longue (en bloc).
cite
Cette balise permet de faire référence à une source externe.
dd
Cette balise permet de décrire un objet dans une liste descriptive.
dfn
Cette balise permet de caractériser une définition.
dl
Cette balise définit une liste descriptive.
dt
Cette balise définit un object à décrire dans une liste descriptive.
figcaption
Cette balise décrit le contenu de sa balise mère <figcaption>.
figure
Cette balise désigne un contenu à part qui accompagne le reste du document.
kbd
Cette balise permet de définir du qui a été tapé avec un clavier.
meter
Cette balise permet de définir une valeur mesurée parmi une liste de valeurs fractionnelles numériques dans un intervalle spécifique, tel qu'un pourcentage ou une note sur 20 par exemple.
progress
Cette balise permet de mesurer le progrès d'une tâche en cours.
q
Cette balise définit une citation courte (en ligne).
time
Cette balise définit une date ou une heure.
var
Cette balise définit une variable mathématique.

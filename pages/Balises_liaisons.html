Les balises de liaisons

a

Balise permettant de mettre des liens internes, externes et un lien vers une ancre

audio

Pour inserer des fichiers audios dont le fichier est sur le WEB ou sur le disque interne

img

Permet d'inserer une image dans une page , dont le fichier est sur le WEB ou sur le disque interne

link

Permet de lier un document HTML à une feuille de styles CSS

objet

Pour inserer n'importe quel type d'objet multimedia

video

Permet d'inserer des fichiers videos dont le fichier est sur le WEB ou sur le disque interne

source

Cette balise est ultiliser pour inserer des fichiers dont le fichier est sur le WEB ou sur le disque interne

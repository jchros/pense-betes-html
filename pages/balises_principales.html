Les balises principales

body
Cette balise contient le contenu de la page qui sera affiché.
!DOCTYPE html
Cette balise se met au début d'un fichier HTML. Il indique au navigateur que le fichier doit être interprété en langage HTML 5.
<!-->
Cette balise permet d'ajouter un commentaire, situé entre <!-- et -->.
html
Cette balise indique au navigateur le code à interpréter en HTML.
head
Cette balise sert d'en-tête au document HTML: dans cet en-tête figurera des éléments charactéristiques de la page tel que son nom, ou l'encodage à suivre pour interpréter du texte.
meta
Cette balise permet d'indiquer des méta-données sur le document.
link
Cette balise lie un document HTML à une feuille de style CSS ou un code JavaScript.
title
Cette balise indique au navigateur le nom du document, à afficher sur une barre d'onglets par exemple.
script
Cette balise permet d'insérer du code JavaScript, permettant d'effectuer des opérations sur le contenu du document.
style
Cette balise permet d'insérer du code CSS, permettant de modifier la mise en page du document.

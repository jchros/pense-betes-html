Les balises principales

Cette page définit les balises suivantes:

!DOCTYPE html

Se met au début d'un fichier HTML. Il indique au navigateur que le fichier doit être interprété en langage HTML 5.

<!-->

Permet d'ajouter un commentaire, situé entre <!-- et -->.

html

Balise indiquant au navigateur le code à interpréter en HTML.

head

Balise servant d'en-tête au document HTML: dans cet en-tête figurera des éléments charactéristiques de la page tel que son nom, ou l'encodage à suivre pour interpréter du texte.

meta

Balise permettant d'indiquer des méta-données sur le document.

link

Lie un document HTML à une feuille de style CSS ou un code JavaScript.

title

Balise indiquant au navigateur le nom du document, à afficher sur une barre d'onglets par exemple.

script

Permet d'insérer du code JavaScript, permettant d'effectuer des opérations sur le contenu du document.

style

Permet d'insérer du code CSS, permettant de modifier la mise en page du document.

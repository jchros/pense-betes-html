Les balises de structuration:

article
Cette balise identifie un contenu dont l'information est cohérente.
aside
Cette balise identifie à un contenu en lien avec un dl ou un section mais dont le contenu est connexe.
caption
Cette balise représente le titre d'un tableau.
div
Cette balise identifie des zones de la page auxquelles une mise en forme spécifique doit être associée, par contre elle n'a aucune sémantique! Il s'agit de l'équivalent en bloc de <span>.
footer
Cette identifie le pied de page d'une zone qui peut être la page elle-même.
h1...h6
Ces balises définissent des titres et leur taille(h1...h6).

<h2>Ceci est un grand titre</h2>

Ceci est un grand titre

<h6>Celui-ci est très petit</h6>

Celui-ci est très petit
header
Cette balise identifie l'en-tête d'une zone (ne pas confondre avec <head>) qui peut être la page elle-même.
li
Cette balise permet d'insérer les items de la liste.
main
Cette balise représente le contenu principal du <body> du document ou de l'application.
menu
Cette balise permet de définir des options spéciales quand l'utilisateur fait un clic droit sur la zone concernée.
nav
Cette balise contient un menu (pouvant être dans le header ou pas).
ol
Cette balise permet de définir des listes numérotées.
section
Cette balise fait correspondre une zone (de la page ou elle-même à l'intérieur d'une zone).
span
Cette balise identifie une zone dans un texte auquelle une mise en forme spécifique doit être associée, par contre elle n'a aucune sémantique! Il s'agit de l'équivalent en ligne de <div>.
style
Cette balise permet d'insérer du code CSS directement dans la page HTML.
table
Cette balise permet de définir un tableau.
tbody
Cette balise représente le corps d'un tableau HTML.
tfoot
Cette balise permet de définir le pied d'un tableau.
thead
Cette balise définit l'ensemble des lignes qui définit l'en-tête des colonnes d'un tableau.
td
Cette balise représente les données d'un tableau.
tr
Cette balise représente l'élément d'une ligne d'un tableau.
th
Cette balise représente les cellules d'en-tête d'un tableau.
ul
Cette balise permet de définir une liste à puce.
